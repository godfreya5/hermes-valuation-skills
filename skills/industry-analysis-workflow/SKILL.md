# 行业分析七步工作流

> 源自肖璟《如何快速了解一个行业》的核心方法论，整合为从定位到输出的完整流程。

## 工作流概览

```
Step 1: 定位生命周期 ──→ Step 2: 护城河 ──→ Step 3: 竞争格局
         │                      │                   │
         └──────────────────────┼───────────────────┘
                                ▼
Step 4: PEST外部因素 ──→ Step 5: 景气度追踪 ──→ Step 6: 估值判断
                                                            │
                                                            ▼
                                              Step 7: 金字塔输出（含终局视角）
```

**贯穿全流程**: AI辅助研究——每步用AI协助数据搜集和假设验证，但不由AI做最终判断。

## I — 方法论骨架 (Interpretation)

这个七步工作流的核心思想是：**不同产业生命周期阶段的行业，分析重点截然不同**。不能用一个模板套所有行业——煤炭行业的核心逻辑是供给收缩，新能源汽车的关键是产销超预期。

首先用渗透率定位行业所处阶段（<15%导入期/15-40%成长期/>40%成熟期），然后按阶段动态分配分析权重：导入期重点看商业模式能否跑通，成长期重点看市场规模能长多大，成熟期重点看护城河和竞争格局能不能守住利润。

在基本面判断之后，引入外部因素（PEST）判断催化剂或阻力，再通过景气度指标的高频数据验证中长期判断，最后用生命周期匹配的估值框架做出投资或决策判断。整个分析用金字塔原理组织输出，"终局视角"倒推当前行动——模糊的正确优于精确的错误。

AI全程作为研究助手：用假设驱动的思路指挥AI搜集数据、拆解验证，但最终的判断和提炼由人完成。

## A1 — 书中的应用案例 (Past Application)

### 案例 1: 新能源汽车行业完整分析（第9章）
- **定位生命周期**: 渗透率从个位数飙升至30%+，处于成长期，重点看市场规模
- **护城河**: 比亚迪的资源垄断（垂直一体化供应链）+ 特斯拉的网络效应（超充网络+FSD用户锁定）
- **竞争格局**: 造车新势力价格战白热化，横向格局分散→正在集中；纵向格局中电池厂话语权强
- **PEST**: 政策（补贴退坡→双积分）、经济（消费降级影响高端车）、文化（环保意识）、技术（固态电池）
- **景气度**: 月度销量/渗透率/电池级碳酸锂价格作为先行指标
- **估值**: 成长期用成长股估值逻辑（高PE容忍+终局折现）

### 案例 2: 智能手机行业的完整产业生命周期演示（第1章）
- 2007年iPhone发布，渗透率<5%导入期
- 2010年iPhone 4发布，渗透率突破20%进入成长期
- 2014年渗透率>65%，行业进入成熟期，"没有创新""挤牙膏"成为常态
- 验证了渗透率15-40%阈值的有效性

1. 想了解某个新兴行业的前景和投资价值（如"人形机器人赛道怎么样"）

## E — 可执行步骤 (Execution)

当skill被激活后，按以下七步流程执行。每次执行应根据行业特点调整各步深度——导入期行业Step2-3简化，Step6用VC估值替代；成熟期行业Step1可简化，Step2-3深化。

### Step 1: 定位生命周期

**目标**: 用渗透率判断行业处于哪个产业生命周期阶段。

1. **计算渗透率**: 渗透率 = 存量用户 ÷ 潜在客户群
2. **定位阶段**:
   - < 15% → **导入期**（分析重点：商业模式可行性）
   - 15% ~ 40% → **成长期**（分析重点：市场规模空间）
   - > 40% → **成熟期**（分析重点：护城河和竞争格局）
   - 渗透率持续下降 → **衰退期**（关注替代品威胁和供给收缩速度）
3. **用AI辅助**: 搜索"XX行业渗透率2024/2025"获取最新数据，交叉验证多个来源
4. **完成标准**: 明确判断行业处于哪个阶段 + 给出渗透率数据来源 + 判断是否有"第二增长曲线"的可能

### Step 2: 护城河分析

**目标**: 判断行业龙头能否持续防御竞争。

1. **分析资源垄断**: 检查生产要素独占——劳动力(稀缺人才)、土地(稀缺资源/牌照)、资本(规模优势)、技术(专利壁垒)、数据(独占数据)
2. **分析网络效应**: 检查生产关系独占——与政府关系(特许经营)、与同行关系(标准制定)、与供应商关系(独家供应)、与客户关系(高转换成本)
3. **关键问题**: 品牌和成本优势是护城河的**结果**而非**原因**——追问品牌靠什么维持、成本低靠什么支撑
4. **完成标准**: 列出行业2-3条核心护城河 + 每条护城河的构成要素 + 判断可持续性（10年维度）
5. **AI辅助**: 搜索"XX行业龙头市场份额演变"判断护城河是否在强化

### Step 3: 竞争格局分析

**目标**: 理解同行怎么分蛋糕 + 上下游怎么分利润。

1. **横向格局**:
   - 计算CR3/CR5行业集中度
   - 判断市场结构: CR3<30%分散竞争 / 30-50%低集中竞争 / 50-80%低集中寡占 / >80%极高寡占
   - 分析集中度趋势：产业生命周期中先降后升（成长期大量新进入者→成熟期淘汰出清）
2. **纵向格局**:
   - 画出产业链价值分配图
   - 分析每个环节的议价权（参考波特五力）
   - 识别产业链上利润最丰厚的环节
3. **产能/库存周期辅助判断**:
   - 资本性支出÷折旧摊销 < 1.5 → 产能底部
   - 外来者涌入扩产能 → 接近顶部
   - 库存天数处于历史高位 → 供过于求
4. **完成标准**: 给出行业内前5名公司市场份额 + 产业链价值分配描述 + 判断竞争激烈程度及趋势

### Step 4: PEST外部因素

**目标**: 找出可能引爆或摧毁行业的外部力量。

1. **政治(P)**: 产业政策方向（扶持性:口头→补贴→直投→强制 / 限制性:准入→供给→价格→受众）、国际关系影响
2. **经济(E)**: 经济周期位置、货币三价格（购买力/通胀→利率→汇率）对行业的影响
3. **社会文化(S)**: 人口结构变化、消费习惯变迁、文化趋势
4. **技术(T)**: 替代品威胁、技术迭代速度、创新层级（基础层→中间层→应用层是否已成熟）
5. **完成标准**: 四个维度各找到至少1个实质性影响因素，判断综合是偏正面还是偏负面

### Step 5: 景气度追踪

**目标**: 用高频数据验证或修正中长期判断。

1. **产业链法**: 找到产业链核心驱动环节（通常是瓶颈环节），从其景气推断上下游
2. **行业关键指标法**（按行业类别差异化）:
   - 上游（资源/材料）→ 看价格+库存
   - 中游（制造/加工）→ 看产销量+产能利用率
   - 必选消费 → 看CPI+社零增速
   - 可选消费 → 看房地产周期+居民收入
   - 服务业 → 看宏观经济指标
   - TMT → 看自身产业周期
3. **完成标准**: 列出3-5个核心景气度指标 + 最新数据 + 趋势判断（向上/向下/拐点）

### Step 6: 估值判断

**目标**: 用生命周期匹配的估值框架判断当前估值合理性。

1. **用相对估值排除宏观干扰**: 行业PE ÷ 全市场PE = 相对估值水平
2. **按生命周期匹配估值逻辑**:
   - 导入期 → VC估值法（看梦想/潜在空间/团队/技术壁垒）
   - 成长期 → 成长股逻辑（终局折现 / PEG / 高PE容忍）
   - 成熟期 → 周期股逻辑（看供需拐点 / 产能周期位置）
3. **交易拥挤度预警**: 换手率/成交额占比处历史高位 → 情绪过热
4. **完成标准**: 判断当前估值"偏高/合理/偏低" + 给出核心逻辑 + 拥挤度信号

### Step 7: 金字塔综合输出

**目标**: 将分析结果组织成逻辑自洽的判断。

1. **罗列**: 把六步分析的全部发现列出
2. **分组**: 按主题归类（可按Step 1-6分组，也可按多空因素分组）
3. **提炼**: 每组用一句话概括核心发现 —— 追问"So What?"直到产生信息增量
4. **归纳**: 用一句话给出行业核心判断（"这个行业值得/不值得投资/进入，因为……"）
5. **终局视角**: 设想行业5-10年后的稳态格局，从终局倒推今天的行动
6. **完成标准**: 输出包含: 核心判断(一句话) + 3-5个支撑论点 + 终局推演 + 关键风险

## B — 边界 (Boundary) ★

### 不要在以下情况使用此skill

- 纯公司个股分析（这是公司研究而非行业研究，虽然行业是背景，但此时应聚焦公司层面）
- 短期交易决策（本框架是中长周期分析工具，不适用于日内/周度交易）
- 用户只需要一个数据或名词解释（简单信息查询无需启动全流程）
- 宏观大势研判（如"中国经济会怎样"——这是宏观研究层面）
- 已经进入衰退期且无可挽回的行业（如实体唱片——框架仍适用但需告知用户行业的客观衰退现实）

### 作者在书中警告的失败模式

- **模板化陷阱**: 对所有行业套同一个模板——煤炭行业和新能源的核心逻辑完全不同
- **线性外推误区**: 用营收增速判断生命周期——口罩行业疫情期增速飙涨不代表进入成长期
- **确认偏误**: 只找支持自己预设观点的信息——尤其投资前已持仓时
- **渗透率误判**: 15-40%阈值不是铁律——B2B行业/基础设施行业的渗透率逻辑不同
- **景气度低时的悲观线性外推**: 行业低谷期只看负面高频数据形成"杀估值"恶性循环
- **价值陷阱**: 衰退期看着PE很低就以为便宜——可能是盈利持续下滑的前夜

### 作者的盲点 / 时代局限

- 框架以消费互联网和制造业经验为主，对AI原生行业（大模型/Agent）的生命周期形态可能不完全适用
- 麦肯锡咨询背景偏向"服务企业决策"，个人投资者执行全套分析的可行性和必要性需打折
- A股市场案例为主，跨境/全球行业分析需额外调整
