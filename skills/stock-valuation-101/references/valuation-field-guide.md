# 估值框架实战扩展

> 基于多只股票的公式驱动估值实战总结。补充 `stock-valuation-101` 主 skill 未覆盖的边界情形。

## 一、五公式执行流程

每次估值按此顺序执行，不可跳步：

```
公式1(DCF) → 公式2(分子分母归因) → 公式3(PE) → 公式4(ROE) → 公式5(FCF趋势)
                                        ↓
                                  三角交叉验证矩阵
                                        ↓
                                   赔率计算
                                        ↓
                                   决策函数
```

## 二、停用条件——以下情形禁用估值框架

| 情形 | 判定标准 | 示例 |
|------|---------|------|
| ST/退市风险 | PE=N/A + 股价<2元 + BPS<1元 | ST海王 |
| 壳股/妖股 | 无主营 + 无利润 + 无FCF + PB极高 | 韩建河山 |
| 重组未完成 | 注入资产未披露 + 历史财务无参考价值 | 石药景峰 |
| 早期亏损 | 导入期 + 商业模式未验证 + 无盈利时间线 | 和元生物 |
| 纯概念/信仰 | PB>10x + ROE<3% + PE>100x | 有研硅 |

停用时直接输出：「不可估值，不在框架内」，不要强行套公式。

## 三、周期股调整

船舶、航运、钢铁等强周期行业，PE 在低谷时极高（误导性看空）、高峰时极低（误导性看多）。需调整：

| 替代指标 | 用法 |
|---------|------|
| PB | 周期股核心指标。PB<1.5x=底部，PB>3x=顶部（视行业调整） |
| 周期均利PE | 用过去5-8年平均利润计算，消除单年波动 |
| 订单/产能周期 | 订单排期是利润的先行指标，领先利润2-3年 |
| 产品价格指数 | 新船价格、运价指数等高频先行指标 |

## 四、SOTP 分部估值（多元化集团）

当公司有多块性质不同的业务时，整体PE失真。用 SOTP：

```
Step 1: 拆——盈利用PE、亏损用PS、现金直接加
Step 2: 估——每块找对标公司，保守给倍数
Step 3: 加——汇总 ÷ 总股本
```

典型适用场景：小米（手机+IoT+互联网+造车）、复星、华润。

## 五、多股对比框架

同时分析多只股票时，用统一表格对比：

| 维度 | 权重 |
|------|------|
| PE | 越低越好，但区分周期股和成长股 |
| PB | 越低越好，提供下行保护 |
| ROE | >15%优秀，<8%平庸 |
| 利润趋势 | QoQ/YoY方向最重要 |
| 护城河/确定性 | 定性判断，决定估值中枢 |
| 赔率 | +0.3以上才有意思 |

## 六、溢价归因陷阱

当一只股票 PB/PE 远高于同行时，必须**精确归因**溢价来源，不能笼统归为"概念炒作"：

- 永鼎股份：溢价来自光纤超级周期 + CPO + 超导，不能简单说"超导溢价"
- 深科技：溢价来自 HBM 封装期权，同行对比才能量化
- 有研硅：溢价来自国产替代信仰，对标全球龙头（信越3x PB vs 有研13x PB）

**原则**：归因时列出所有溢价来源，逐一评估合理性，用同行对比量化溢价幅度。

## 七、港股 vs A股同源资产对比

同一集团在不同市场上市，估值可能差数倍：

| 方法 | 
|------|
| 同一集团资产，买便宜的那个市场 |
| PE/PB 对比量化溢价差 |
| 确认资产注入明细后再估值（重组股） |

典型案例：石药集团（HK 15x PE）vs 石药系A股（PE N/A + 数倍PB）。
