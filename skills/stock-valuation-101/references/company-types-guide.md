# 公司类型适配指南

从 10+ 实盘估值案例中归纳的分类体系。

## 成长型（科技/消费龙头）
**主公式**: PE + ROE + 情景推演 | **次**: PB
- PE 在历史中位偏高可以接受（成长溢价）
- 重点验证 ROE 趋势和 FCF 质量
- 例: 紫光股份（PE 37x, ROE 10%, 新华三护城河）

## 周期型（船舶/重工/资源）
**主公式**: PB + 订单/产能景气度 | **次**: 周期调整 PE
- 单年 PE 完全失效——低谷时极高，高峰时极低
- 用「周期平均 EPS × 中周期 PE」替代单年 PE
- 例: 中国船舶（PB 3x, 手持订单排至 2028）

## 项目驱动型（基建/管材/B2G）
**主公式**: PB + BPS 底线 | **次**: 周期平均 PE
- 利润极不均匀，PE 经常为 N/A
- BPS 提供硬底部——PB < 1.5x 才有安全边际
- 例: 青龙管业（PB 1.9x, BPS 7.5 元），双鹭药业（PB 1.26x, BPS 4.98 元）

## 概念溢价型（HBM/CPO/超导/国产替代）
**主公式**: 赔率 + 对标估值 | **次**: DCF
- PE 极高，定价包含远期期权
- 关键是拆解「基本面估值 vs 概念溢价」
- 赔率 < 0 且上行空间 < 20% → 风险回报不对等
- 例: 深科技（PE 68x, HBM 溢价 ~40%），有研硅（PB 13.5x, 国产替代溢价）

## 不可估值型
**触发条件**: PE > 200x 或为负 + ROE < 3% 或为负 + 无 BPS 支撑 + 无护城河
- 果断停止——不在框架射程内
- 例: 永鼎（PE 307x, Q4 亏损, Q1 同比 -45%），韩建河山（壳股），ST海王
